# Sudoku-Inspired Logic App

A Python project that explores computational logic and algorithmic problem-solving through Sudoku-style puzzles.

## Project Goal
The goal of this project is to implement a logic-based game engine using Python, focusing on:
* Algorithmic problem-solving.
* Efficient data structures for grid management.
* Logical constraints validation.

## Tools Used
* **Language:** Python
* **Development Environment:** Carnets (iOS) / VS Code
